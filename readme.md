# Data Science Learning & Experimentation Repository

## Overview

This repository contains my complete **Data Science learning journey and experimentation workspace**, covering concepts from **Python fundamentals to Machine Learning, Deep Learning, Data Visualization, Web Scraping, and Automation Projects**.

The purpose of this repository is to build a **strong practical foundation in Data Science and AI development** while maintaining a structured codebase of experiments, practice exercises, and mini-projects.

This repository serves as:

- A **structured learning archive**
- A **practice environment for experimentation**
- A **portfolio of mini projects**
- A reference implementation for **data science workflows**
